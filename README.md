# meu-portifolio-qa
Portfólio QA

1. Visão geral do projeto
Testes automatizados com Cypress para aplicação de e-commerce
Tecnologias usadas (Cypress, cypress-mochawesome-reporter, cypress-multi-reporters)

2.Estrutura do repositório
automacao — scripts e configuração do Cypress
documentos — casos de teste, plano de teste e relatório de bugs
evidencias — evidências dos testes

3. Como configurar
Entrar em automacao
Instalar dependências: npm install

4. Como executar os testes
Executar em modo aberto: npx cypress open
Executar em modo headless: npx cypress run

5. Localização dos testes
spec.cy.js

6. Relatório e evidências
Relatório HTML: index.html
Screenshots: screenshots
Documento de evidências: evidencias.md

7. Documentação de QA
Casos de teste: casos-de-testes.md
Plano de testes: plano-de-testes.md
Relatório de bugs: relatorio-de-bugs.md
